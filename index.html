Abstract

Our teams objective is to create a handheld water depth sensor that will interface with a phone to give the user the depth, for use in applications such as canyoneering, fishing, cliff diving, search and rescue.

We need to utilize either ultrasonic sensors or sonar in order to calculate water depth regardless on how clear the water is. Needs to be small enough to be handheld, lightweight for taking in a pack.

This project allows improved safety in several applications including those mentioned above. Knowing how deep or shallow water is can prevent accidents from happening in areas.
